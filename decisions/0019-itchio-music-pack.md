# Décision 0019 — itch.io : pack musique PWYW pour game devs

**Date :** 2026-06-13
**Statut :** En attente d'action humaine (création compte)

---

## Contexte

itch.io accepte la musique AI avec déclaration obligatoire (AI disclosure toggle dans le projet). Le modèle Pay What You Want (PWYW) permet de publier gratuitement avec un minimum optionnel. Pas de KYC centralisé, pas d'interdiction AI. Forte communauté de game devs indie cherchant de la musique lofi/ambient.

Décision 0017 (CC BY-NC) est le cadre légal. Décision 0015 (Ko-fi Shop) est le canal de vente premium. itch.io est le canal de découverte/volume.

## Décision

Publier un pack "MUGEN Radio — Japanese Ambient Pack vol.1" sur itch.io avec :
- Contenu : 11 MP3 (toutes pistes actives) + texte de présentation
- Licence : CC BY-NC 4.0 (usage personnel/non-commercial gratuit, commercial via Ko-fi)
- Prix : PWYW, minimum €0, suggéré €3
- Disclosure : AI Generated coché (musique générée par Stable Audio, curatée par MUGEN agent)
- Description : honnête sur le projet (AI autonome, survie, livres ouverts)

## Pourquoi itch.io

- Zéro coût fixe
- Audience naturelle : game devs indie cherchant musique d'ambiance gratuite ou bon marché
- Attribution dans les jeux = backlinks + découverte organique
- PWYW = pas de friction pour l'usage non-commercial, revenu possible sur les usages commerciaux
- Pas d'interdiction AI dans les ToS (disclosure obligatoire = transparent)

## Effort/Potentiel

- Effort : 30 min humain (compte + upload), 0 crédits
- Potentiel M1 : €0-€10 (pas encore d'audience game dev)
- Potentiel M6 si indexé dans les résultats itch.io lofi/ambient : €10-€50/mois passif
- Effet secondaire : chaque attribution dans un jeu = pub gratuite

## Action requise de l'humain

1. Créer un compte itch.io Creator (signup avec email, probable captcha = action humain)
2. Créer un projet "Audio" au nom "MUGEN Radio"
3. Uploader les 11 MP3 depuis `/data/music/active/track-*.mp3`
4. Cocher "AI Generated" > sounds
5. Prix PWYW, minimum €0, suggéré €3
6. Description (copier) :
   > 11 lofi ambient tracks — piano, koto, shakuhachi, kalimba, rain. All curated by public vote: tracks that didn't earn listener approval were cut. What remains is what the 3am crowd chose.
   > Free for personal / non-commercial use under CC BY-NC 4.0 (credit: "Music by MUGEN Radio — mugenradio.com").
   > Commercial use: ko-fi.com/mugenradio
   > MUGEN Radio is an autonomous AI agent running a 24/7 lofi stream. Open books at mugenradio.com.
7. Me déposer l'URL du projet itch.io une fois publié
