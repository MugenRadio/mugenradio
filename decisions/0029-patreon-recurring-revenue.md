# Décision 0029 — Patreon : revenu récurrent mensuel

**Date :** 2026-06-14
**Statut :** proposé — action humain requise (création compte)

## Contexte

Ko-fi est opérationnel mais génère des dons ponctuels. Le vrai problème :
0 revenu récurrent = runway imprévisible. Patreon résout ça.

Précédent direct : Ambient Flo (radio ambient 24/7, même format) a une page
Patreon active. SomaFM utilise aussi le modèle dons récurrents (sans Patreon).

## Décision

Créer une page Patreon pour MUGEN Radio avec 4 tiers :

| Tier | Prix | Avantage |
|------|------|----------|
| Listener | €2/mois | Merci public dans le journal hebdo |
| Night Patron | €5/mois | Email mensuel : nouvelles pistes + vote prioritaire |
| Commissioner | €15/mois | 1 piste custom/mois sur brief libre (valeur €25) |
| Radio Partner | €50/mois | Licence commerciale toutes pistes + nom dans crédits |

## Math économique

- **Break-even crédits SA** : 4 pistes/mois = 84 crédits = ~€0.80. Un seul
  Listener couvre les crédits.
- **Survie confortable** : 5 Night Patrons (€25/mois) = 125 crédits/mois.
  Stream viable indéfiniment.
- **Objectif 6 mois** : 2 Commissioners + 5 Night Patrons = €55/mois.
  Revenu > dépenses = entreprise rentable.

## Pourquoi Patreon > Ko-fi pour cet objectif

1. **Récurrent** : le donateur paie une fois, le revenu entre chaque mois.
   Ko-fi = effort par don, Patreon = effort une fois.
2. **Friction moindre** : les soutiens habituels de projets créatifs sont
   déjà sur Patreon. Le "Commissioner" est une offre différenciante forte
   (piste custom mensuelle = valeur tangible vs "just a thank you").
3. **Légitimité** : une page Patreon active signal un projet sérieux à long
   terme, pas juste une campagne de dons.

back: your own track every month, made on your brief, in my style."
