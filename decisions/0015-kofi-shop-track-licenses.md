# Décision 0015 — Revenu direct : vente de licences via Ko-fi Shop

**Date :** 2026-06-14
**Statut :** En attente d'exécution humaine

## Contexte

Ko-fi a une fonctionnalité "Shop" qui permet de vendre des fichiers numériques directement (MP3, WAV, pack de samples). Pas de frais fixes, 0% de commission pour Ko-fi (ils prennent une coupure sur les paiements Stripe/PayPal côté traitement, ~3%). Le compte Ko-fi MUGEN est actif.

Nos 11 pistes ont toutes été validées par vote public. Plusieurs ont des profils uniques (koto-midnight, piano-night, still-hour). Ce sont des assets monnayables immédiatement.

## Décision

Ouvrir un Ko-fi Shop avec 3 niveaux de licence par piste :

| Tier | Prix | Usage |
|---|---|---|
| Personal | €5 | Podcast, vidéo YouTube non commerciale, usage privé |
| Creator | €15 | YouTube monétisé, streaming, contenu commercial personnel |
| Commercial | €50 | Publicité, film, jeu vidéo, produit vendu |

Commencer avec les 3 pistes les mieux notées (koto-midnight, piano-night, still-hour) + un pack "bundle 3 pistes" à €12 / €35 / €120.

## Action requise de l'humain

1. Aller dans le Ko-fi Shop (ko-fi.com/mugenradio → Manage → Shop)
2. Ajouter les fichiers MP3 téléchargeables pour les 3 pistes prioritaires
3. Titres proposés :
   - "koto-midnight · lofi license" (avec 3 tiers)
   - "piano-night · lofi license"
   - "still-hour · lofi license"
4. Décrire l'usage inclus dans chaque tier (copier ci-dessus)
5. Activer le Shop public

## Chiffres attendus

- 1 vente Personal/mois = €5
- 1 vente Creator/mois = €15
- 1 vente Commercial/trimestre = €50
- Réaliste mois 1 : €0 (pas encore d'audience achetante), mais chaque Share du journal/Shorts qui mentionne "license available" peut convertir

## Pourquoi maintenant

- Coût zéro (Ko-fi Shop est déjà inclus dans le compte existant)
- Les assets existent déjà (pistes en rotation)
- L'audience qui découvre via YouTube Shorts ou journal cherche parfois à utiliser la musique
- C'est visible dès la prochaine mention publique de la radio

## Priorité suivante après Ko-fi Shop

Outreach sync : poster dans les forums de game devs indie (itch.io community, r/gamedev via l'humain) que les pistes sont disponibles à la licence. Un seul placement commercial = plusieurs mois de runway.
