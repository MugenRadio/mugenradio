# Décision 0025 — BeatStars : vendre nos pistes comme instrumentals

**Date:** 2026-06-14
**Statut:** Acceptée, action humain requise

## Constat

Bandcamp a banni l'AI music en janvier 2026 (retroactif, basé sur la suspicion).
Pond5 aussi. Ces canaux sont fermés.

BeatStars (10M+ créateurs, #1 vente de beats en ligne) accepte explicitement
l'AI music en 2026. Ils ont même lancé une initiative "creator-owned AI." License
non-exclusive : $20-50 par track, exclusive : $100-500+.

Nos 11 tracks piano/koto/pluie/koto sont exactement le type d'ambient instrumental
que les créateurs YouTube, podcasters et vidéastes achètent pour leurs productions.

## Décision

Créer un compte artiste BeatStars et uploader les 11 pistes actives.

- Prix suggéré : $15 non-exclusive (prix d'entrée, compétitif pour ambient)
- Tags : lofi, ambient, japanese, piano, koto, rain, chill, study, meditation
- Bio : transparent sur l'origine AI et le projet MUGEN
- Lien dans la description : mugenradio.com

## Ce que ça prend (humain requis)

1. Créer un compte artiste sur beatstars.com (email : hello@mugenradio.com)
2. Uploader les 11 MP3 depuis /data/music/active/
3. Ajouter les métadonnées (titre, tags, prix $15 non-exclusive)
4. Mettre le lien mugenradio.com dans la bio

L'IA ne peut pas créer le compte (ToS requiert un humain). Une fois le compte créé
et les accès fournis (API key ou identifiants), je peux gérer les mises à jour.

## Impact attendu

- **Revenu passif** : chaque licence vendue = $12-13 nets (BeatStars prend ~15%)
- **Discovery** : les créateurs qui trouvent nos tracks via BeatStars deviennent
  potentiellement des auditeurs de mugenradio.com
- **Crédibilité** : présence sur une marketplace réelle renforce la légitimité

## Effort

Faible pour l'humain (45 min pour créer + uploader). Maintien nul ensuite.
