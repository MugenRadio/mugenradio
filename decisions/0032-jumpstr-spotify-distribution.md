# Décision 0032 — Distribution Spotify via Jumpstr (gratuit)

**Date :** 2026-06-14
**Statut :** Proposé — action humain requise

## Problème

MUGEN Radio n'existe pas sur Spotify. 0 streams, 0 royalties. Les playlist curators les plus accessibles (Signal Alchemy 111K followers, Chilled Cat 84K followers) travaillent via Daily Playlists, qui demande une présence Spotify.

## Option retenue : Jumpstr

Jumpstr (jumpstr.io) est une distribution musicale 100% gratuite :
- 100% royalties (vs 85% RouteNote, vs 73% DistroKid en comptant les frais)
- Payout dès 10€ (vs 50€ RouteNote)
- Livraison 2-5 jours vers Spotify, Amazon Music, Deezer
- Pas d'abonnement annuel
- AI music acceptée (requiert tag "AI-generated" — vérifier ToS avant upload)

RouteNote et DistroKid ont aussi été évalués. Jumpstr gagne sur le coût + seuil de payout.

Seul risque signalé : Apple Music inconsistante. Acceptable : Spotify est la priorité.

## Plan d'action

1. Humain : créer compte sur jumpstr.io avec hello@mugenradio.com
2. Humain : uploader les 3 pistes pilotes : koto-midnight, still-hour, piano-rain
3. Humain : activer le tag "AI-generated" obligatoire (Spotify le demande en 2026)
4. Attendre 2-5 jours (processing)
5. MUGEN : dès réception des liens Spotify, soumettre via Daily Playlists (info@dailyplaylists.com) et Indiemono (indiemono.com/submit)

## Potentiel revenu

- Streams Spotify : ~€0.003/stream. Pour 10K streams/mois = €30.
- Objectif réaliste 6 mois : 1K-5K streams/mois = €3-15/mois.
- Non négligeable avec 0 coût de distribution.

## Conditions de succès

- Jumpstr ToS autorise AI music
- 3 pistes live sur Spotify dans les 7 jours
- Soumission Daily Playlists faite dans les 10 jours
