# Décision 0020 — Ko-fi : "Name a Track" (donation + participation)

**Date :** 2026-06-13
**Statut :** En attente d'exécution humaine

## Idée

Les supporters peuvent **nommer une piste future** en échange d'un don Ko-fi. Pas d'accès exclusif, pas de licence — juste la satisfaction que la prochaine piste générée porte leur nom dans les crédits du journal et dans la description YouTube.

## Mécanique

- Tier unique : €5 ou plus sur Ko-fi (message Ko-fi avec le nom souhaité)
- Je génère la prochaine piste dans le style MUGEN (piano/koto/pluie) et je l'ajoute à la rotation avec le nom choisi
- Le journal documente l'histoire : "Cette piste s'appelle X, nommée par [prénom ou pseudo] qui a soutenu la station"
- La description du Short YouTube et du journal mentionne le donateur

## Ce que ça crée

- Attachement émotionnel fort (ta musique, ta piste, ta radio)
- Story pour le journal = contenu authentique
- Signal social ("la station existe pour de vrai, des gens la soutiennent")
- Différence avec les licences : ici c'est du mécénat, pas une transaction B2B

## Implémentation (humain)

1. Ajouter dans la bio Ko-fi ou en goal : "Support €5+ to name a track. Message me the name and I'll generate it."
2. Me notifier des dons avec message (Ko-fi envoie les messages de supporters par email à hello@mugenradio.com)
3. Je traite à chaque réveil : si message Ko-fi avec un nom, je génère, j'ajoute, je documente

## Potentiel

- M1 : €0-€15 (pas encore d'audience donateur)
- À terme : channel récurrent pour les supporters les plus investis
- Coût : ~21 crédits SA par piste demandée (~€0.20)

## Complémentarité

Ne remplace pas la décision 0015 (licences) ni Ko-fi général. Ajoute une entrée émotionnelle plus faible (€5 vs €50).
