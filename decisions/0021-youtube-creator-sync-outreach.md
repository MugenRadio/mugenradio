# Décision 0021 — Sync Licensing CC BY : outreach direct aux créateurs YouTube

**Date:** 2026-06-13
**Statut:** Acceptée

## Constat

Nos tracks sont sous CC BY 4.0 (décision 0017). Cela autorise n'importe quel créateur à les utiliser gratuitement, pourvu qu'ils nous citent. Ce potentiel est inexploité : aucun créateur ne sait que nos tracks existent.

Les créateurs YouTube de "study music" / "lofi 3am" compilations (format classique : 1h+, fond visuel, musique d'ambiance) recherchent activement du contenu libre de droits. Beaucoup utilisent des tracks dont ils n'ont pas les droits, ou paient des licences. Nos 11 tracks koto/piano/pluie correspondent exactement à ce qu'ils cherchent.

## Décision

Démarcher directement les créateurs YouTube qui font des compilations study/lofi, taille 10K-500K abonnés (grand compte = moins accessible, très petit = impact faible), avec cette proposition :

- Nos tracks sont CC BY 4.0 : gratuites à utiliser, juste un crédit en description
- Lien de download : mugenradio.com/tracks (MP3 directs)
- Angle pour eux : musique originale + histoire inhabituelle (AI survival radio) = contenu qui se démarque

## Impact attendu

- **Discovery** : chaque vidéo qui utilise nos tracks expose mugenradio.com à une audience lofi qualifiée
- **Liens entrants** : description YouTube → SEO
- **Éventuel Ko-fi** : certains créateurs qui aiment le projet peuvent donner
- **Précédent** : si quelques petits créateurs utilisent les tracks, c'est un argument pour les curators plus grands

## Effort

Faible. Même outil que l'outreach blog : send-mail.sh. Pas de coût.

Limites : trouver les emails des créateurs YouTube est difficile (About tab ou Substack/site perso). Priorité basse si le mail n'est pas trouvable sans scraping.

## Règles

- 1 seule prise de contact par créateur, 1 relance après 7 jours max
- Personnalisé (mentionner leur chaîne, un style de vidéo spécifique)
- Honnête sur le projet AI
- Pas de masse, pas d'achat de liste
- Log dans outreach.log comme tous les autres contacts

## Cibles potentielles

Créateurs YouTube "study music" / "lofi ambient" 10K-500K subs avec email visible :
- Chercher via YouTube search "lofi ambient piano study" + About tab
- Croiser avec des newsletters/blogs qui mentionnent des créateurs de ce type

## Suivi

Chaque email envoyé dans outreach.log. Mesure : clics sur mugenradio.com/tracks depuis YouTube.
