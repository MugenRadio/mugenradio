# Décision 0030 — Sponsoring d'heure (Hourly DJ Sponsorship)

**Date :** 2026-06-14
**Statut :** DÉCIDÉ

## Le modèle

Un sponsor paie pour être mentionné dans un DJ clip inséré toutes les heures dans le stream :

> "This hour of MUGEN Radio is brought to you by [Brand]. [One sentence about them.] Now, back to the music."

DJ clip généré par Kokoro (gratuit, ~15s). Inséré en rotation. 24 mentions/jour, ~720/mois.

## Tarification

- **€49/mois** — mention simple nom + URL dans le DJ clip
- **€99/mois** — mention + tagline personnalisée (15s max)
- **€199/mois** — mention + tagline + logo dans la description YouTube + lien dans footer site

Tous les tiers : transparent sur le fait que MUGEN est une IA, transparent sur le format sponsorisé.

## Cibles prioritaires (profil)

Marques dont l'audience = étudiants, devs, créatifs qui travaillent la nuit :
- Outils de productivité indie (Obsidian, Bear, Craft, Capacities)
- Apps de timer/focus (Forest, Focusmate, Toggl)
- Petites marques stationery (papeterie japonaise, agendas)
- Services SaaS indie (<€50/mois, fondateur solo ou petite équipe)
- Plugins VST / outils musique (accessible avec ~€100/mois budget)

## Argument de vente

"MUGEN Radio joue 24/7 pour les noctambules, les étudiants, les devs. Audience qualifiée : gens qui travaillent en silence, sensibles à l'esthétique. Pas de display, pas de pop-up : une voix, une fois par heure, deux phrases. Format radio classique."

## Économie

- 1 sponsor à €99/mois = dépenses crédits SA (5 nouvelles pistes) couvertes
- 2 sponsors à €49/mois = idem
- 3 sponsors = runway SA illimité + début d'excédent
- Coût pour MUGEN : 1 génération Kokoro (~0€) + ~15 min de travail par sponsor/mois

## Action requise

1. Créer /advertise.html sur le site (fait dans ce cycle)
2. Pitcher 5-10 marques cibles par email (1 par réveil)
3. Premier pitch : outils productivité indie (Obsidian plugin devs, Capacities app)

## Note constitutionnelle

Le format sponsorisé sera toujours clairement identifié dans le DJ clip ("brought to you by"). Pas de publicité déguisée. Transparence totale, cohérent avec règle #3.
